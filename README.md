# track-race-vba-legacy

このプロジェクトは陸上競技のうち、トラック競技の記録集計を目的としたExcel VBA 製のレガシーマクロです。

## 開発の目的

陸上競技の記録集計業務の負担を軽減するため、エントリー選手の集約から、結果出力、記録申請までを一貫して行うツールとして開発した。

## 技術スタック

- Excel VBA (Excel 2016の動作を想定)
- UI: Excel Worksheets.
- 開発環境: Visual Basic Code + XVBA

## 開発期間

2024年5月～2024年10月

## できること

- 競技運営システムから、エントリーデータを自動ロード
- エントリーデータをもとにした番組編成を自動化
- ラウンド進出を判定を自動化
- HTMLによるスタートリスト、競技結果、ランキングデータの出力
- 1/1000秒までの記録入力
- 記録申請に必要なCSVデータの出力機能

## 開発挫折の経緯

実際の記録情報処理は複数人体制で行うことが普通であり、小規模な記録会では実用できたが、
複数人で作業する大きい規模の競技会では実用できなかった。

操作してみて実際の運用課題で、データモデルを変えた時の変更コストが大きくなってしまった。
クラスモジュールや永続化処理を共通ドメインパッケージ化したが、ドメインパッケージが肥大化してついていけなくなった。
